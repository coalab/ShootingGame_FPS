# 탑다운 슈팅 게임 (Phaser 3)

Phaser 3로 만든 2D 탑다운 슈팅 게임입니다. CDN 방식으로 별도 설치 없이 브라우저에서 바로 실행할 수 있습니다.

## 실행 방법

1. `index.html` 파일을 브라우저에서 엽니다.  
   - 파일 탐색기에서 더블클릭하거나  
   - 브라우저 주소창에 `index.html` 경로를 입력합니다.
2. 별도의 서버나 빌드 없이 바로 플레이할 수 있습니다.

## 웹에서 실행하기 (클릭 한 번에 실행)

**게임이 바로 실행되려면 GitHub Pages를 사용하세요.** HTML Preview는 스크립트 실행 제한으로 게임이 동작하지 않을 수 있습니다.

### 1) GitHub Pages로 실행 링크 만들기

1. GitHub에서 해당 저장소 열기 → **Settings** → 왼쪽 메뉴 **Pages**
2. **Source**에서 **Deploy from a branch** 선택
3. **Branch**에서 `main`(또는 사용 중인 브랜치), 폴더는 **/ (root)** 선택 후 **Save**
4. 1~2분 뒤 아래 주소로 접속하면 게임이 바로 실행됩니다.

   **실행 링크:** `https://[본인GitHub아이디].github.io/[저장소이름]/`

   예: 아이디가 `myid`, 저장소 이름이 `topdown-shooter` 이면  
   → **https://myid.github.io/topdown-shooter/**

### 2) README에서 클릭으로 열기

저장소 주소가 정해지면, README에 아래처럼 **실제 링크**를 넣어두면 클릭 한 번에 게임이 실행됩니다.  
(아래는 예시이므로 본인 주소로 바꿔 넣으세요.)

```markdown
[▶ 게임 바로 실행](https://본인아이디.github.io/저장소이름/)
```

## 조작법

| 입력 | 동작 |
|------|------|
| **방향키** (↑ ↓ ← →) | 플레이어 이동 |
| **마우스** | 플레이어가 마우스 커서 방향을 바라봄 |
| **마우스 클릭** | 클릭한 방향으로 발사 |
| **스페이스바** | 현재 바라보는 방향으로 발사 |

## 게임 규칙

- **플레이어**: 녹색 삼각형. 적을 처치하고 적의 공격을 피하며 점수를 쌓습니다.
- **적**: 빨간 원. 화면 가장자리에서 등장해 플레이어를 추적하며, 주기적으로 주황색 총알을 발사합니다.
- **점수**  
  - 적 처치: **+10**  
  - 적과 접촉: **-5**  
  - 적 총알에 맞음: **-15**  
  - 점수는 0 미만으로 내려가지 않습니다.

## 요소 설명

| 구분 | 설명 |
|------|------|
| **플레이어 총알** | 노란색. 적에게 맞으면 적이 제거됩니다. |
| **적 총알** | 주황색. 플레이어에게 맞으면 점수가 감점됩니다. |
| **배경** | 어두운 회색 격자 배경 (800×600). |

## 기술 스택

- **Phaser 3** (v3.70.0) – CDN 링크 사용
- **HTML5** – 단일 `index.html`에 HTML, CSS, JavaScript 포함
- **Arcade Physics** – 이동, 충돌 처리
